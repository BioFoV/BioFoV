BioFoV - Integrated forensic video analyzer

BioFoV is a free, cross-platform program for forensic experts, that eases tedious video analysis and brings to the normal human being algorithms that so far have only been available to programmers and scholars.

This project is currently being moved to this repository and will soon be publicly available, for now this repository serves only as an issue tracker and a user manual.
